## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Zero dependencies
- Highly performant
- Mobile-first responsive design

## Deploy

A landing page é estática e pode ser hospedada em qualquer CDN ou servidor estático:

- Vercel
- Netlify
- AWS S3 + CloudFront
- GitHub Pages
- Cloudflare Pages

## Desenvolvimento

```bash
# Apenas abra o index.html no navegador
open index.html

# Ou use um servidor local
npx serve .
```
