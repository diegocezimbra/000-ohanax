// Design Pattern: Strategy
interface PaymentStrategy {
  pay(amount: number): Promise<Receipt>;
}

class PixPayment implements PaymentStrategy {
  async pay(amount: number) {
    const qrCode = await generatePix(amount);
    return { method: 'pix', qrCode };
  }
}

Perguntas Frequentes
Qual o formato do curso?

São 66 seções em HTML interativo com navegação wizard. Cada seção contém teoria profunda, diagramas, código multi-linguagem, exercícios práticos e um quiz de 10 questões. Seu progresso é salvo automaticamente no navegador.

Preciso saber alguma linguagem específica?

Não. Os exemplos usam TypeScript, Python, Rust, Go e outras linguagens, mas o foco é nos conceitos e padrões — que são universais. Se você programa em qualquer linguagem, vai aproveitar 100% do conteúdo.

O conteúdo é atualizado?

Sim. O curso cobre tecnologias e práticas atuais incluindo AI Engineering, RAG, Prompt Engineering, FinOps, Platform Engineering, Post-Quantum Cryptography e Computação Quântica.

Posso estudar no celular?

Sim. Todas as páginas são totalmente responsivas e funcionam em qualquer dispositivo — desktop, tablet ou celular.

Tem certificado?

O foco é no conhecimento real, não em certificados. Ao completar todas as 66 seções com nota máxima nos quizzes, você terá um domínio que fala por si em qualquer entrevista técnica.
