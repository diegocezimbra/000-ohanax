Proposta de Valor: App móvel para checklist diário de motoristas e veículos, garantindo compliance com legislação e reduzindo riscos operacionais e jurídicos para frotas de qualquer tamanho.

2. O Produto

Funcionalidades Core

  • Checklist do Motorista: CNH (validade, categoria), exame toxicológico, atestado médico
  • Checklist do Veículo: CRLV, pneus, freios, luzes, extintor, triângulo, macaco
  • Registro Fotográfico: Fotos obrigatórias de itens críticos com geolocalização
  • Assinatura Digital: Motorista assina confirmando inspeção
  • Alertas Automáticos: CNH vencendo, CRLV atrasado, exame toxicológico
  • Dashboard Web: Gestor visualiza compliance de toda frota
  • Relatórios: Exportação para auditoria e fiscalização

Diferenciais

  • 100% offline - funciona em áreas sem sinal
  • Integração com DENATRAN para validação automática de CNH
  • OCR para leitura automática de documentos
  • Templates customizáveis por tipo de operação

Cálculos

LTV = ARPA x Margem Bruta x Tempo Médio (meses)
LTV = R$1.500 x 85% x 36 meses = R$45.900
LTV/CAC = R$45.900 / R$5.000 = 9,2x (meta: >3x)
CAC Payback = R$5.000 / (R$1.500 x 85%) = 3,9 meses (meta: <12 meses)
Conclusão: Unit economics saudável. LTV/CAC de 9,2x indica negócio escalável com boa margem para investir em aquisição.
