SMART NOTER - DOCUMENTAÇÃO TÉCNICA COMPLETA EM HTML
====================================================

Esta pasta contém a documentação técnica completa do Smart Noter
dividida em módulos HTML para navegação fácil e organizada.

ESTRUTURA DOS ARQUIVOS:
=======================

1. index.html - Página principal com visão geral
2. design-system.html - Design System completo
3. arquitetura.html - Arquitetura de Software  
4. stack-tecnologico.html - Stack e dependências
5. banco-de-dados.html - Schemas e configuração DB
6. apis-endpoints.html - APIs REST completas
7. pipeline-ia.html - Pipeline de IA (Whisper, GPT-4)
8. implementacao.html - Código de implementação
9. seguranca.html - Segurança e proteções
10. devops.html - Docker, K8s, CI/CD
11. testes.html - Estratégia de testes
12. guia-desenvolvimento.html - Cronograma 10 semanas

13. styles.css - CSS compartilhado

COMO USAR:
==========

1. Abra index.html no navegador
2. Navegue pela sidebar ou links
3. Cada página tem navegação prev/next
4. Todo código tem syntax highlighting
5. Design responsivo para mobile
